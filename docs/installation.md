# Установка RUSH CLI

Для установки RUSH CLI выполните следующие шаги:

1.  **Клонирование репозитория**

    Откройте терминал и клонируйте репозиторий RUSH CLI с GitHub:

    ```bash
    git clone https://github.com/Karasichek/rush-cli.git
    ```

2.  **Запуск установочного скрипта**

    Перейдите в директорию проекта и запустите скрипт `SETUP.sh`:

    ```bash
    cd rush-cli
    chmod +x SETUP.sh
    ./SETUP.sh
    ```

    Скрипт `SETUP.sh` выполнит следующие действия:
    *   Обновит список пакетов вашей системы.
    *   Установит необходимые зависимости, такие как `tmux`, `helix`/`micro`, `rsync`, `eza`, `fastfetch`, `git`, `lazygit`, `lf`, `xmake` и другие. Если `lazygit` недоступен в пакетном менеджере, он будет установлен из официального репозитория.
    *   Скопирует конфигурационные файлы RUSH CLI в вашу домашнюю директорию, обеспечивая симметричную настройку для `zsh`, `bash` и `sh`.

3.  **Перезапуск оболочки**

    После завершения установки перезапустите вашу оболочку или выполните команду `reload` (если алиас уже доступен):

    ```bash
    source ~/.zshrc  # Для Zsh
    source ~/.bashrc # Для Bash
    source ~/.shrc   # Для Sh
    ```

    При первом запуске оболочки (если файл `$HOME/FLAG` отсутствует) будет произведена дополнительная проверка и установка пакетов, что гарантирует наличие всех необходимых инструментов для комфортной работы с RUSH CLI.
